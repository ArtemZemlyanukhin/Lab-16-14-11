## 🔧 Реализованные функции

- ### 1. `double* full_elements(double* ptr_array, int size);`
- **Назначение**: Заполнение массива значениями
- **Параметры**: 
  - `ptr_array` - указатель на массив
  - `size` - размер массива
- **Возвращает**: указатель на заполненный массив
### 2. `double* calc_elements(double* ptr_array, int size);`
- **Назначение**: заполнение отрицательных значений массива нулями
- **Параметры**: 
  - `ptr_array` - указатель на массив
  - `size` - размер массива
- **Возвращает**: указатель на заполненный массив
- ### 3. `int put_elements(double* ptr_array, int size);`
- **Назначение**: вывод на экран элементов массива
- **Параметры**: 
  - `ptr_array` - указатель на массив
  - `size` - размер массива
- **Возвращает**: указатель на заполненный массив
- ### 4. `int delete_k(double* ptr_arr, int size, int k, int count);
- **Назначение**-удаление некоторого количества элементов массива,начиная с позиции k
- **Параметры**: 
  - `ptr_arr` - указатель на массив
  - `size` - размер массива
  - `k` - индекс начала удаления
  - `count` - количество удаляемых элементов 
- **Возвращает**: новый размер массива после удаления
- ### 5. `double* insert(double* ptr_arr, int* size, int index, double num);`
- **Назначение**-Вставка элемента в массив по указанному индексу
- **Параметры**: 
  - `ptr_arr` - указатель на массив
  - `size` - указатель на размер массива
  - `index` - индекс для вставки
  - `num` - значение для вставки
- **Возвращает**: указатель на массив после вставки
- ### 6. `double* insert_after_min_k(double* ptr_arr, int* size, int k, double num);`
- **Назначение**-вставка -999 после минимального числа массива
- **Параметры**: 
  - `ptr_arr` - указатель на массив
  - `size` - указатель на размер массива
  - `k` - шаг вставки
  - `num` - значение для вставки
- **Возвращает**: указатель на массив после вставки
- ### 7. `int find_min_index(double* ptr_arr, int size);`
- **Назначение**-поиск индекса с наименьшим числом
- **Параметры**: 
  - `ptr_arr` - указатель на массив
  - `size` - указатель на размер массива
- **Возвращает**: индекс минимального элемента при успехе, -1 при ошибке
- ### 8. `double task_14(int size, int A);`
- **Назначение**-Поиск порядкового номера минимального значения элементов,
больших заданного значения А
- **Параметры**: 
  - `size` - размер массива
  - `A` - заданное значение
- **Возвращает**: значение минимального элемента, большего А при успехе, 0 при ошибке
- ### 9. `double task_for_lab11(int size);`
- **Назначение**-Среднее арифметическое элементов массива, расположенных
между первым и вторым отрицательными элементами
**Параметры**: 
  - `size` - размер массива
- **Возвращает**: Значение среднего арифметического при успехе, 0 при ошибке
- ### 10. `struct ArrayData create_random_array();`
- **Назначение**-Создание массива со случайными числами
- **Возвращает**-структуру ArrayData с случайным массивом
- ### 11. `struct ArrayData create_d_array(struct ArrayData a, struct ArrayData b, struct ArrayData c);`
- **Назначение**: Создание массива d по правилу di=max(ai,bi,ci)
- **Параметры**: 
  - `a` - указатель на массив
  - `b` - указатель на массив
  - `c` - указатель на массив
- **Возвращает**: структуру ArrayData c новым массивом d
- ### 12. `int printf_all_arrays();`
- **Назначение**-вывод массивов a,b,c,d
- **Возвращает**-0 при успехе, 1 при ошибке
- ### 13. `int* full_elements_for17(int* ptrarr, int size);`
- **Назначение**: Заполнение массива целых чисел значениями по формуле
- **Параметры**:
  - `ptrarr` - указатель на массив целых чисел
  - `size` - размер массива
- **Возвращает**: указатель на заполненный массив
- ### 14. `void sort_bubble(int* ptrarr, int n);`
- **Назначение**: Сортировка массива методом пузырька (Bubble Sort)
- **Параметры**:
  - `ptrarr` - указатель на массив целых чисел
  - `n` - размер массива
- **Возвращает**: ничего (сортировка выполняется на месте)
- ### 15. `void sort_select(int* ptrarr, int n);`
- **Назначение**: Сортировка массива методом выбора (Selection Sort)
- **Параметры**:
  - `ptrarr` - указатель на массив целых чисел
  - `n` - размер массива
- **Возвращает**: ничего (сортировка выполняется на месте)
- ### 16. `int func(int x);`
- **Назначение**: Вычисление значения по специальной формуле
- **Параметры**:
  - `x` - входное целое число
- **Возвращает**: вычисленное значение по формуле (четные: x*10*3, нечетные: (x*10)^2)
- ### 17. `void swp(int* ptrarr, int* ptrarr2);`
- **Назначение**: Обмен значениями двух переменных
- **Параметры**:
  - `ptrarr` - указатель на первую переменную
  - `ptrarr2` - указатель на вторую переменную
- **Возвращает**: ничего (значения меняются местами)
- ### 18. `void sort_qwi(int* ptrarr, int n);`
- **Назначение**: Быстрая сортировка массива (Quick Sort)
- **Параметры**:
  - `ptrarr` - указатель на массив целых чисел
  - `n` - размер массива
- **Возвращает**: ничего (сортировка выполняется на месте)
